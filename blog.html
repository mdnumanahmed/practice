Question and Answer

1. What is DOM and what is the purpose of DOM?

HTML ফাইল হচ্ছে একটি Document. যখন HTML document ব্রাউজারে লোড হয়, তখন এটি document object হয়ে যায়। জাভাস্ক্রিপ্ট এই Document এর একটি Object Model তৈরি করে। আমরা ডট নোটেশনের মাধ্যমে প্রতিটি নোডে একসেস করতে পারি। যেমন- document.body HTML ফাইল এর মধ্যে যতগুলো ট্যাগ আছে, জাভাস্ক্রিপ্ট সব ট্যাগকে Node হিসেবে বিবেচনা করে একটি Tree তৈরি করে নেয়। যাকে Document Object Model বলা হয়।

DOM Methods দ্বারা আমরা JavaScript দিয়ে HTML elements গুলোকে manipulate (add, create, delete, remove, modify etc.) করতে পারি।

2. How will you select HTML elements using DOM? Name the DOM methods.

DOM দিয়ে HTML ফাইল এর কোন element কে ট্যাগের নাম, ক্লাসের নাম, আইডি ব্যবহার করে ধরতে পারি। নিম্নে কিছু মেথডের নাম দেয়া হলো।

  • getElementById()
  • getElementsByClassName()
  • getElementsByTagName()
  • querySelector()
  • querySelectorAll()
3. What is event bubble?

HTML ফাইলের কোন ইলিমেন্টে ক্লিক করলে, রুট ইলিমেন্ট থেকে শুরু করে সর্বনিম্ন লেভেল পর্যন্ত (উপর থেকে নিচে যেখানে ক্লিক করা হয়েছে) event খুঁজতে থাকে, যাকে Capturing Phase-1 বলে।
তারপর যেখানে ক্লিক করা হয়েছে তারমধ্যে কোন event থাকলে ঐ ইলিমেন্টকে trigger করে, তাকে Target Phase 2 বলে।
তারপর উপরের দিকে যেতে যেতে খুঁজতে থাকে আর কোন event আছে কি না? কোন event পাওয়া গেলে trigger করে দেয়, তাকে Bubbling Phase 3 বলে।

4. What is a callback function and why will you use it?

যখন কোন ফাংশনের আর্গুমেন্ট হিসেবে আরেকটি ফাংশন পাঠানো হয়, তখন মূল ফাংশনটি এক্সিকিউট হওয়ার পর এ ফাংশনটি এক্সিকিউট হয়, তখন আর্গুমেন্ট পাঠানো ঐ ফাংশনটিকে callback function বলে।
